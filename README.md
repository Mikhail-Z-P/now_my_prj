# Проект now.my.prj

## Описание:
Цель: Этот проект представляет собой виджет для обработки банковских операций

Что он делает:  
1. Маскирует номера карт и счетов;  
2. Преобразует ISO-даты в формат «ДД.ММ.ГГГГ»;  
3. Фильтрует транзакции по статусу (state);  
4. Сортирует транзакции по дате (date).
## Установка:
1. Убедитесь, что у вас установлен Python версии 3.8 или выше.     
2. Склонируйте репозиторий с помощью команды git clone <https://github.com/Mikhail-Z-P/now_my_prj>.  
3. Перейдите в директорию проекта: cd <now.my.prj>     
4. Установите необходимые зависимости: pip install -r requirements.txt     

## Использование:
