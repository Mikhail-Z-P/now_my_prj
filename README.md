# Проект now.my.prj

## Описание:
Цель: Этот проект представляет собой виджет для обработки банковских операций

Что он делает:  
1. Маскирует номера карт и счетов;  
2. Преобразует ISO-даты в формат «ДД.ММ.ГГГГ»;  
3. Фильтрует транзакции по статусу (state);  
4. Сортирует транзакции по дате (date).

## Генераторы проекта
В проекте реализованы следующие генераторы для обработки данных:

Модуль с генераторами [generators.py](src%2Fgenerators.py)

`filter_by_currency` — генератор для фильтрации транзакций по валюте. Принимает список транзакций и код валюты,   
возвращает только те транзакции, где валюта совпадает с указанной.

`transaction_descriptions` — генератор для извлечения описаний транзакций. Принимает список транзакций и последовательно возвращает описания каждой операции.

`card_number_generator` — генератор для создания номеров карт. Формирует номера карт в заданном диапазоне в формате XXXX XXXX XXXX XXXX.
## Декораторы проекта 
Добавлен декоратор log для логирования выполнения функций.

Назначение:

Фиксирует начало выполнения функции, её успешное завершение или ошибку;   
выводит лог в консоль или сохраняет в файл (по выбору).
## Установка:
1. Убедитесь, что у вас установлен Python версии 3.8 или выше.     
2. Склонируйте репозиторий с помощью команды git clone <https://github.com/Mikhail-Z-P/now_my_prj>.  
3. Перейдите в директорию проекта: cd <now.my.prj>     
4. Установите необходимые зависимости: pip install -r requirements.txt
